# Housing-Price-Prediction 🏠
This repository contains a project for predicting housing prices using regression models on a dataset from Kaggle.

## DataSet
- Source: Kaggle (Housing Price Dataset)
   * Path: https://www.kaggle.com/datasets/yasserh/housing-prices-dataset
- The dataset includes features such as price, area, bedrooms, and other house attributes.

## Models Implemented
Three regression models were used to predict housing prices:

1. Linear Regression
2. Ridge Regression
3. Lasso Regression

## Results
- Linear Regression and Ridge Regression were the best-performing models.
- Lasso Regression performed reasonably but was a little bit less accurate.
- Evaluation metrics included MSE, RMSE and R² score.
